# YiteWang.github.io

Personal academic website for Yite Wang, built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll template.

## Local Development

Install Ruby/Bundler dependencies, then run:

```sh
bundle install
bundle exec jekyll serve
```

The site is configured for a GitHub Pages user site at `https://YiteWang.github.io` with an empty `baseurl`.

## Content

- Homepage: `_pages/about.md`
- Publications: `_bibliography/papers.bib`
- News: `_news/`
- Education/CV data: `_data/cv.yml`
- Service page: `_pages/service.md`
- Profile image: `assets/img/prof_pic.jpg`
